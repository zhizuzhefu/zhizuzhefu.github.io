FAQ / 常见问题

为什么文件无法导入？ / Why can't I import a file?

请先确认文件格式是否受支持（如 MP3、WAV、M4A 等常见音频格式均可导入），并确保文件大小未超过 500 MB。如果文件来自其他 App，请尝试通过"共享"菜单将文件发送到 AudioTools。若问题仍然存在，请重启 App 后再试。 / First, check that your file is in a supported format — common audio formats like MP3, WAV, and M4A can all be imported. Also ensure the file is under 500 MB. If the file is from another app, try using the Share menu to send it to AudioTools. If the issue persists, restart the app and try again.

为什么 WiFi 传输无法访问？ / Why can't I access WiFi Transfer?

请确认你的电脑和 iOS 设备连接到同一个 WiFi 网络，然后在浏览器中输入 App 内显示的本地地址（如 http://192.168.x.x:8080）。如果仍无法连接，请检查路由器是否开启了"客户端隔离"功能，或尝试关闭 VPN 后重试。 / Make sure your computer and iOS device are connected to the same Wi-Fi network, then enter the local address shown in the app (e.g., http://192.168.x.x:8080) into your browser. If it still doesn't connect, check whether your router has "client isolation" enabled, or try disabling any VPN and retrying.

为什么没有收到通知？ / Why didn't I receive notifications?

请前往"设置 > 通知 > AudioTools"，确认通知权限已开启。如果你使用了专注模式，请确保 AudioTools 未被屏蔽。即使关闭通知，所有核心转换功能仍可正常使用，不受影响。 / Go to Settings > Notifications > AudioTools and make sure notifications are enabled. If you're using Focus mode, check that AudioTools is not blocked. Rest assured that all core conversion features work perfectly even with notifications turned off.

支持哪些输出格式？ / What output formats are supported?

AudioTools 支持导出为以下格式：M4A、AAC、MP3、WAV、CAF（Apple Lossless）、Ogg + Opus 以及 MP4。你可以根据使用场景选择合适的格式——例如 MP3 兼容性最广，WAV 适合无损需求，M4A 在苹果设备上表现最佳。 / AudioTools supports exporting to M4A, AAC, MP3, WAV, CAF (Apple Lossless), Ogg + Opus, and MP4. Choose the format that best fits your needs — MP3 for maximum compatibility, WAV for lossless quality, or M4A for the best experience on Apple devices.

保存公开素材，整理成自己的本地素材库

影存帮助你从分享文本或公开链接识别图片、视频和实况素材。确认你拥有保存和使用权后，素材会保存在设备本地，并可通过标签、搜索、裁切比例记录和视频封面提取继续整理。

Save public media you have rights to use

YingCun is a local-first media saver and organizer. Paste or share a public link, review the detected resources, confirm usage rights, then keep images, videos, and Live Photo-style pairs in your on-device library.

Core Flow / 核心流程
简体中文
  1. 从其他 App 分享文本或链接到影存，或在收件箱粘贴链接。
  2. App 在本机解析公开资源，展示可保存的图片、视频或实况素材。
  3. 保存前确认你拥有使用权；保存后用标签和搜索整理素材。
English
  1. Share text or a link into YingCun, or paste a link in Inbox.
  2. The app locally resolves public resources and previews available images, videos, or Live Photo-style pairs.
  3. Confirm usage rights before saving, then organize assets with tags and search.
Organization / 素材整理
简体中文
  • 标签、无标签筛选和搜索帮助你快速找回素材。
  • 可记录裁切比例，方便后续发图、剪辑或复用。
  • 视频可以提取封面，实况素材以静态图和动态视频成组管理。
English
  • Tags, untagged filters, and search help you find saved assets quickly.
  • Crop-ratio notes keep future publishing and reuse more predictable.
  • Extract covers from videos; keep Live Photo-style still and motion files grouped.
Supported Sources / 支持范围
简体中文

首版面向公开直链、抖音短视频、小红书公开笔记和 X.com 公开帖子。解析规则可以通过公开 JSON 配置更新，App 内解析和下载仍在本机执行。对用户主动打开的本机浏览器会话，App 可列出当前网页已暴露的 http/https 图片、视频或 HLS 候选。

登录页、私密内容、付费墙、DRM、blob/data URL、加密流、平台限制或版权保护内容不会被绕过。

English

The first release targets direct public media URLs, Douyin short videos, Xiaohongshu public notes, and X.com public posts. Resolver rules can be updated through public JSON config while parsing and download remain local. In a user-initiated local browser session, the app can list http/https image, video, or HLS candidates already exposed to the current page.

Login walls, private content, paywalls, DRM, blob/data URLs, encrypted streams, platform restrictions, and copyright protection remain restricted.

English
  • Assets, tags, and edit metadata stay on your device by default.

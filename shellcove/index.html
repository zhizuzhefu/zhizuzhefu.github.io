Privacy as Architecture / 隐私是架构
简体中文
  • 配对走 X25519 ECDH，daemon 永远不知道你手机的私钥，反之亦然
  • 每一帧 WebSocket 都用 AES-256-GCM 用配对密钥加封
  • SSH 凭据、配对密钥、密码全部只存在你设备的 iOS Keychain
  • Daemon 由你自己托管，没有 Shellcove 账号系统
English
  • Pairing uses X25519 ECDH; the daemon never learns your phone's private key and vice versa
  • Every WebSocket frame is sealed with AES-256-GCM using the paired key
  • SSH credentials, pairing keys, and saved passwords live only in the iOS Keychain on your device
  • You host the daemon. We don't. There is no Shellcove account.
Quickstart / 快速开始
简体中文

在你的 Mac 或 Linux 开发机上：

$ brew install shellcove/tap/shellcoved $ shellcoved init $ shellcoved pair

终端会显示一个二维码和 6 位配对码，App Store 安装 Shellcove，扫码即配对完成。

English

On your Mac or Linux machine:

$ brew install shellcove/tap/shellcoved $ shellcoved init $ shellcoved pair

The terminal prints a QR code and a 6-digit pairing code. Install Shellcove from the App Store, scan, and you're paired.

Requirements / 系统要求
简体中文
  • iOS 17 及以上的 iPhone 或 iPad
  • Mac 或 Linux 上运行开源 shellcoved 守护进程（单二进制）
  • 手机和 daemon 间的网络可达性 — 同一 Wi-Fi、Tailscale、ZeroTier、Cloudflare Tunnel、frp 都可以
  • 不提供中继，所有连接直连
English
  • iOS 17 or later, iPhone or iPad
  • A Mac or Linux machine running the open-source shellcoved daemon (single binary)
  • Network reachability between phone and daemon — local Wi-Fi, Tailscale, ZeroTier, Cloudflare Tunnel, frp, or anything else you trust
  • No relay provided. All connections are direct.
